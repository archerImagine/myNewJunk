The 'People' search function is intended to help you find people using criteria such as:
  • First or Last Name, UserID, Phone Number
  • Location, Organization, Job Title
  • Keywords (searches against 'Additional Information' and 'Expertise fields' in users' profiles)
People Search can query for Expertise terms, however if you are looking to find people with specific subject matter expertise, use the newly-enabled Experts search to achieve more targeted and relevant results.
Experts search allows you to find subject matter experts that match the terms you enter here, which could include:
  • Skill, Technology, Language
  • Product, Customer, Vertical
  • Role
Example search terms: telepresence, Advanced Services, Java, project management, user experience, fluent French

Learn more about searching for Experts and Expertise.
Directory Framework
You may use the Directory application in either of two modes: Full or Light.
The Full mode is like that of a regular CEC page and contains more functionality, such as the tabs on the left side of the page to access Navigation and My Links and other dynamic page content.
The Light mode has less functionality but may load faster than the Full version. Users over dial-up or slow connections may choose to use the Light version to improve response times.
You may switch between versions at any time.
Hover Card Display Options
In the Directory application, the Hover Card appears whenever you mouse over a linked name with a dotted underline. This pop-up style window gives you a preview of a person's profile without needing to navigate to their profile page. This is useful when browsing a list of search results.
This preference allows you to turn the Hover card on or off.
Search Results Display Options
These options control default behaviors for searches that you run in Directory. Although these changes can always be made in Directory itself, this pane gives you one place to control all of your configurable options.
The first preference allows you to control how many search results you see in each page.
The second preference allows you to show or hide people's photos in the search results listing.
The third preference allows you to control whether a person's profile displays every level in an employee's reporting chain on individual lines or a collapsed, shorter version, showing many levels in one line.
